TransitLink is a smart transport planning web application that helps users find the best routes across bus, rail, and ferry services, while prioritising accessibility and user preferences.
## Key Features
### Smart Route Search
•	Find routes instantly 

•	Compare travel options 
### Accessibility Filtering
•	Wheelchair access 
•	Step-free routes 
•	Audio announcements 
•	Priority seating 
### AI Chat Assistant
•	Ask about delays 
•	Get help with routes 
•	Understand your options 
 ### Saved Routes & Places
•	Home, school, work, shopping 
•	Quick access to frequent journeys 
### Dashboard
•	Recommended routes 
•	Personalised experience





## How to run it:
1.	Download the TRANSITLINK.zip that can be found on the GitHub repository.
2.	Once the files in the Zip file have been downloaded, open app.py which can be found under the main file. 
3.	In the console of app.py, run python app.py using the file path where the file has been saved.
4.	If it does not run, check that flask has been installed on VS Code to ensure it runs. 
5.	Copy the link that is given once python app.py runs. (The link should be http://127.0.0.1:5000 ).
6.	Once the website has loaded, feel free to navigate across the pages available.
